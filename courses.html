JAVA

JAVA

JavaScript (often abbreviated as JS) is a dynamic, high-level programming language standardized as ECMAScript (e.g. ECMA‑262). Originally built to add interactivity to web pages, it is now used widely across platforms—from client-side scripts in browsers to server-side applications using Node.js, as well as mobile apps, games, and desktop tools

REACT JS

REACT JS

React (also called React.js or ReactJS) is a free, open-source JavaScript library developed by Meta for building user interfaces using a component-based, declarative approach. It uses a Virtual DOM to limit costly updates—only changed parts of the UI are re-rendered, leading to better performance.
